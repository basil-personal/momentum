Changelog

A complete history of firmware, app, and API updates. We ship improvements every month.

March 5, 2026
v1.4.0

Firmware: Added support for Mandarin Chinese transcription. Improved noise cancellation algorithm for open-plan offices. Reduced power consumption during idle state by 15%.

App: New timeline view for browsing transcripts by date. Added ability to merge related transcripts from the same meeting. Dark mode now follows system preference automatically.

API: New GET /transcripts/:id/highlights endpoint returns only the highlighted moments from a transcript. Added language filter parameter to GET /transcripts.

February 14, 2026
v1.3.2

Firmware: Hotfix for Bluetooth reconnection issue affecting some Android 14 devices. Resolved rare crash during extended recording sessions exceeding 8 hours.

App: Fixed search results not displaying speaker labels correctly. Improved transcript loading speed by 40% for accounts with more than 500 recordings.

API: Fixed pagination offset error on GET /webhooks. Improved error messages for invalid API key formats.

January 22, 2026
v1.3.0

Firmware: Introduced gesture controls: double-tap to bookmark, long-press to pause, triple-tap to mark private. Added proximity sensor wake for Momentum Personal.

App: Launched smart highlights powered by on-device ML. Added Notion integration for automatic meeting note export. New onboarding flow for first-time users.

API: Added POST /webhooks and DELETE /webhooks/:id endpoints. Webhook payloads now include speaker count and duration metadata.

December 10, 2025
v1.2.1

Firmware: Stability improvements for Conference Wi-Fi connectivity. Fixed LED ring brightness not respecting configured settings after firmware update.

App: Resolved issue where offline transcripts were not syncing after reconnecting to Wi-Fi. Added pull-to-refresh on transcript list.

API: Increased default pagination limit from 10 to 20 items per page. Added device_id filter to GET /transcripts.

November 18, 2025
v1.2.0

Firmware: Added occupancy detection for Momentum Conference. Introduced automatic recording start and stop based on room presence. Improved microphone array calibration on first setup.

App: New device management screen with battery, firmware, and storage indicators. Added Slack integration for sharing transcript highlights. Introduced team directory for automatic speaker naming.

API: Launched GET /devices and POST /devices/:id/config endpoints. API keys can now be scoped to read-only or full access.

October 8, 2025
v1.1.0

Firmware: First major update. Added support for French, German, and Portuguese transcription. Improved battery life by 10% through optimised audio encoding. NFC pairing now works with iOS 16 devices.

App: Launched natural language search across all transcripts. Added speaker identification and voice profile management. New settings screen for recording quality and sync preferences.

API: Initial public API release with GET /transcripts and GET /transcripts/:id endpoints. API key management available in the dashboard.
