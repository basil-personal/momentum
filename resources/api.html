API Reference

Build on Momentum. Access transcripts, manage devices, and integrate conversation intelligence into your own tools.

Overview

The Momentum REST API gives you programmatic access to your transcripts, devices, and account data. All endpoints return JSON and follow standard HTTP conventions. The base URL for all requests is:

https://api.momentum.dev/v1

All requests must be made over HTTPS. HTTP requests will be rejected.

Authentication

Authenticate every request by including your API key in the Authorization header:

Authorization: Bearer mk_live_your_api_key_here

You can generate and manage API keys from the Settings → API section of your Momentum dashboard. Keys are scoped to your account and inherit your account permissions. Never share your API key or commit it to version control.

Endpoints

Transcripts

GET /transcriptsList all transcripts. Supports pagination via ?page= and ?per_page= query parameters. Filter by date with ?after= and ?before=.
GET /transcripts/:idRetrieve a single transcript by ID. Returns the full transcript text, speaker labels, highlights, and metadata.

Devices

GET /devicesList all devices associated with your account. Returns device type, firmware version, battery level, and last sync time.
POST /devices/:id/configUpdate configuration for a specific device. Accepts JSON body with settings such as recording_quality, auto_sync, and led_brightness.

Webhooks

GET /webhooksList all registered webhook endpoints for your account.
POST /webhooksRegister a new webhook endpoint. Accepts url and events array (e.g., ["transcript.completed", "device.synced"]).
DELETE /webhooks/:idRemove a registered webhook endpoint by ID.

Example Request

curl -X GET https://api.momentum.dev/v1/transcripts \
  -H "Authorization: Bearer mk_live_your_api_key_here" \
  -H "Content-Type: application/json"

Response:

{
  "data": [
    {
      "id": "tr_abc123",
      "title": "Product sync — March 14",
      "duration_seconds": 2340,
      "speakers": ["Alice", "Bob", "Carol"],
      "highlights_count": 7,

    }
  ],

    "page": 1,
    "per_page": 20,
    "total": 142
  }
}

Rate Limits

The API allows up to 1,000 requests per minute per API key. If you exceed this limit, requests will return a 429 Too Many Requests response with a Retry-After header indicating how many seconds to wait before retrying.

For high-volume use cases, contact our team about enterprise rate limits.

SDKs

We maintain official SDKs for the most popular languages. Each SDK handles authentication, pagination, error handling, and retries out of the box.

Python

pip install momentum-sdk

Full-featured SDK with async support. Python 3.9+.

Node.js

npm install @momentum/sdk

TypeScript-first SDK with ESM and CJS builds. Node 18+.

Go

go get momentum.dev/sdk

Idiomatic Go client with context support. Go 1.21+.
